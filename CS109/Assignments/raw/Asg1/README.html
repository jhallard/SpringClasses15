This lab0-intro-unix will not be graded and there is no credit for it.  However, it is important you understand the Unix command line and submit procedure before you actually use it in the first assignment.

Make sure you know how to submit files.  Submit these files to lab0.  Verify that the submit actually worked by looking in the submit directory for your username.  The find(1) command is very useful for things like this. 

The submit command copies your files into a directory
/afs/cats.ucsc.edu/class/cmps109-wm.s15/lab0/$USER
and prefixes each file with a sequence number.  You can see the names of the files, but not their contents. 

Prior experience with Unix is assumed.  Attend a lab section to ask questions if you don't understand how submit works. The TA can explain such things. 

Also read the submit checklist: 
http://www2.ucsc.edu/courses/cmps109-wm/:/Syllabus/submit-checklist/

This directory shows a main program and a hello class and illustrates various features of C++11 that will be covered later in the course. 

Whenever you need to look up information about C++, use Google, or look at The C++ Resources Network: 
http://www.cplusplus.com/
